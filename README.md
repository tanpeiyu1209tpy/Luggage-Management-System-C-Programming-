# Luggage Management System (C Programming)

**Year 1 Semester 1 Project**  
Authors: **Tan Pei Yu, Long Lingbo, Liew Chia Ming, Leonard Tze Zi Yang**

A simple Luggage Management System written in C. It allows users to add, view, edit, and remove luggage records, and automatically calculates extra baggage fees based on luggage weight.
